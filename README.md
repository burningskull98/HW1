Анализатор логов
Скрипт на Python для анализа логов доступа Nginx, с акцентом на время запросов и URL'ы. 
Он обрабатывает последний лог-файл, вычисляет статистику по URL'ам и генерирует HTML-отчет с топ URL'ами по суммарному времени запросов.

Функции
Парсит логи доступа Nginx (комбинированный формат с добавленным $request_time).
Поддерживает как обычные .log, так и сжатые .gz файлы логов.
Вычисляет статистику: количество, процент, сумму, среднее, максимум и медиану времени запросов по URL'ам.
Генерирует HTML-отчет с сортируемой таблицей (требуется jquery.tablesorter.min.js).
Настраивается через командную строку или файл конфигурации.
Структурированное логирование в JSON с помощью structlog.
Пропускает обработку, если отчет уже существует.
Обработка ошибок с проверкой порога ошибок парсинга.

Установка
Убедитесь, что установлен Python 3.13+.
Установите Poetry: pip install poetry
Клонируйте или скачайте проект.
Запустите poetry install для установки зависимостей.
